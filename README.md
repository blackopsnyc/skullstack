## Installation
Add this line to your application's Gemfile:

```ruby
gem 'skullstack'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install skullstack
```

## Configuration
Change `application.css` to `application.scss`

Replace the file contents with:

```ruby
@import 'skullstack';
```
